flint_naming - PEP-8 Naming Conventions
=======================================

Check the PEP-8 naming conventions.
This module provides a plugin for the ``flint`` modular code checker.

Installation
------------

You can install, upgrade, uninstall ``flint_naming`` with these commands::

  $ pip install flint_naming
  $ pip install --upgrade flint_naming
  $ pip uninstall flint_naming
